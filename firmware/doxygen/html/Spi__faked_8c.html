Detailed Description

Faked calls record themselves in mock.
Tests access mock to see what the function under test does:

  • calls made by the function under test
  • args passed by those calls.
    TestSuite.exe seg faults if any test accesses mock without defining mock.
    Mock.h declares mock:
    extern Mock_s mock;

mock is a global that is defined in test_runner:

Mock_s *mock;

The memory for mock is allocated and deallocated in the setup/teardown for each test that calls the faked call.
Create/destroy mock for an entire test suite by pointing setUp and tearDown in the test_runner:

setUp = SetUp_Mock; tearDown = TearDown_Mock;

For any test with mock defined:
View all calls made (not an assertion, but helpful for debug)

PrintAllCalls(mock);

Assert a specific call is made

uint16_t call_n = 1;
TEST_ASSERT_TRUE(
AssertCall(mock, call_n, "_SignalDataReady")
);
