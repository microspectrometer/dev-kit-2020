\doxysubsection{Detailed Description}
{\ttfamily usb-\/bridge} communicates over USB via FTDI USB Bridge IC {\ttfamily FT221X} 


\begin{DoxyItemize}
\item USB host sees {\ttfamily FT221X} as a {\bfseries{USB device}}
\item Two options for host to communicate with this USB device\+:
\begin{DoxyEnumerate}
\item use generic USB serial device driver, e.\+g., {\ttfamily pyserial}
\item use the {\ttfamily D2\+XX} driver to access the expanded functionality of the FTDI USB API
\end{DoxyEnumerate}
\item The {\ttfamily usb-\/bridge} microcontroller sees {\ttfamily FT221X} as an {\bfseries{FT1248 slave}}
\end{DoxyItemize}

Flow from USB host to {\ttfamily usb-\/bridge} microcontroller\+:~\newline

\begin{DoxyItemize}
\item USB host writes to {\ttfamily FT221X}
\item now {\ttfamily FT221X} has data in its receive buffer\+:
\begin{DoxyItemize}
\item pin {\ttfamily FT1248 MISO} asserts LOW
\item this signals to the {\ttfamily usb-\/bridge} microcontroller that data is available
\end{DoxyItemize}
\item {\ttfamily usb-\/bridge} microcontroller reads the data
\end{DoxyItemize}

Flow from {\ttfamily usb-\/bridge} microcontroller to USB host\+:~\newline

\begin{DoxyItemize}
\item Control lines
\begin{DoxyItemize}
\item {\ttfamily Ft\+Clock}
\begin{DoxyItemize}
\item the clock idles {\ttfamily LOW}
\item the clock is only active during FT1248 communication
\end{DoxyItemize}
\end{DoxyItemize}
\item Sending a single byte takes three clock cycles
\item TODO\+: multi-\/byte communication where each additional byte only adds one clock cycle
\item Databus pin {\bfseries{direction}}
\begin{DoxyItemize}
\item databus pins are {\ttfamily Ft\+Miosio0} through {\ttfamily Ft\+Miosio7}
\item the databus is bi-\/directional
\begin{DoxyItemize}
\item {\ttfamily Ft\+Read}\+: the {\itshape FT221X} drives the databus
\item {\ttfamily Ft\+Write} the {\itshape microcontroller} drives the databus
\end{DoxyItemize}
\item databus pins are usually microcontroller inputs
\item microcontroller {\bfseries{only outputs to the databus during an {\ttfamily Ft\+Write}}}
\begin{DoxyItemize}
\item the microcontroller pins are outputs for the interval
\item starting after a clock rising edge
\item ending before the next clock rising edge
\end{DoxyItemize}
\end{DoxyItemize}
\item TODO\+: fill this in 
\end{DoxyItemize}

The FT221X indicates data transfer status on Pin {\ttfamily MISO} while FT1248 communication is active.~\newline
{\ttfamily LOW} on pin {\ttfamily MISO} means {\itshape FT221X status is OK.}~\newline

\begin{DoxyItemize}
\item See the {\bfseries{8-\/bit write}} and {\bfseries{8-\/bit read}} timing diagrams\+:
\begin{DoxyItemize}
\item \href{https://www.ftdichip.com/Support/Documents/DataSheets/ICs/DS_FT221X.pdf}{\texttt{ FT221X datasheet}}
\end{DoxyItemize}
\item See ACK/\+NAK details in section {\bfseries{FT1248\+: NAK}}\+:
\begin{DoxyItemize}
\item \href{https://www.ftdichip.com/Support/Documents/AppNotes/AN_167_FT1248_Parallel_Serial_Interface_Basics.pdf}{\texttt{ FTDI application note AN167}}
\end{DoxyItemize}
\end{DoxyItemize}

Timing diagram and ACK/\+NAK summary\+:~\newline

\begin{DoxyItemize}
\item During the {\ttfamily BUS TURNAROUND} after the {\ttfamily COMMAND PHASE}
\begin{DoxyItemize}
\item if command is {\bfseries{8-\/bit read}}
\begin{DoxyItemize}
\item {\ttfamily LOW} indicates {\bfseries{receive buffer}} {\itshape has data} to read
\item {\ttfamily HIGH} indicates {\bfseries{receive buffer}} {\itshape is empty}
\end{DoxyItemize}
\item if command is {\bfseries{8-\/bit write}}
\begin{DoxyItemize}
\item {\ttfamily LOW} indicates {\bfseries{transmit buffer}} {\itshape has room} to write
\item {\ttfamily HIGH} indicates {\bfseries{transmit buffer}} {\itshape is full}
\end{DoxyItemize}
\end{DoxyItemize}
\item During the {\ttfamily DATA} phase
\begin{DoxyItemize}
\item after the clock {\itshape rising} edge
\begin{DoxyItemize}
\item FT221X outputs an {\ttfamily ACK} or {\ttfamily NAK} on {\ttfamily MISO}
\item {\ttfamily ACK}\+:
\begin{DoxyItemize}
\item {\ttfamily MISO} is {\ttfamily LOW}
\item indicates a {\itshape successful} data transfer
\end{DoxyItemize}
\item {\ttfamily NAK}\+:
\begin{DoxyItemize}
\item {\ttfamily MISO} is {\ttfamily HIGH}
\item indicates a {\itshape failed} data transfer
\end{DoxyItemize}
\end{DoxyItemize}
\item after clock {\itshape falling} edge\+:
\begin{DoxyItemize}
\item Microcontroller reads {\ttfamily MISO}
\begin{DoxyItemize}
\item {\ttfamily LOW} indicates {\ttfamily ACK}
\item {\ttfamily HIGH} indicates {\ttfamily NAK}
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyItemize}
\item Data transers {\bfseries{fail}} because of either\+:
\begin{DoxyItemize}
\item a {\itshape buffer full} condition on {\bfseries{writing data}} to the FT221X
\item a {\itshape buffer empty} condition on {\bfseries{reading data}} from the FT221X
\end{DoxyItemize}
\end{DoxyItemize}
