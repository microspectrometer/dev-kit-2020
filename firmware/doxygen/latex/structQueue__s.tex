\doxysubsection{Detailed Description}
Queue uses a byte array as a circular buffer. 

A {\itshape queue} is a byte array with data (head, tail) and methods (push, pop) that access linear memory like circular memory. The memory is finite (the {\itshape queue} can fill up) but the user cannot \char`\"{}fall off\char`\"{} the end of the byte array.~\newline
~\newline
 Picture the byte array as a closed loop like the numbers on a clock\+:
\begin{DoxyItemize}
\item the initial byte is the next byte after the final byte
\item the {\itshape head} and {\itshape tail} are clock-\/hands pointing at the array elements
\item the bytes {\itshape between} head and tail are the bytes in the queue
\end{DoxyItemize}

Other bytes in the buffer are not in the queue, meaning they cannot be accessed by a {\itshape pop}. These values remain stored in the buffer until overwritten by a {\itshape push}.~\newline
~\newline
 A queue is a F\+I\+FO (first-\/in first-\/out) buffer\+: {\itshape pop} removes the oldest byte in the queue.
