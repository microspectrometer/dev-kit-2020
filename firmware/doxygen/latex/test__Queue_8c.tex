\doxysubsection{Detailed Description}
{\itshape Queue tests are coupled}~\newline
 Testing one {\bfseries{Queue}} method requires calling other {\bfseries{Queue}} methods. Lib {\bfseries{Queue}} design intent is that data in a {\ttfamily \mbox{\hyperlink{structQueue__s}{Queue\+\_\+s}}} is only accessible through the {\bfseries{Queue}} methods. The {\ttfamily \mbox{\hyperlink{structQueue__s}{Queue\+\_\+s}}} is intended to be an abstract data type, but it is exposed in {\ttfamily \mbox{\hyperlink{Queue_8h_source}{Queue.\+h}}} to make {\ttfamily inline} possible in the application\textquotesingle{}s translation unit.

{\itshape Empty tests are for generating function doc strings}~\newline

\begin{DoxyItemize}
\item Empty tests contain one line\+: {\ttfamily T\+E\+S\+T\+\_\+\+P\+A\+S\+S();}~\newline

\item Tests start as empty.~\newline

\item If the test is useful, write it using the necessary {\bfseries{Queue}} methods when they are available. 
\end{DoxyItemize}
